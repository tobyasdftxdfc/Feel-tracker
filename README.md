# 🥗 FUEL Tracker — GitHub Pages PWA

A calorie & macro tracker installable to your phone's Home Screen.

---

## 🚀 Deploy in 5 steps

### 1. Create a GitHub account (if you don't have one)
Go to [github.com](https://github.com) → Sign up (free).

### 2. Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `fuel-tracker`
3. Set it to **Public**
4. ✅ Check **"Add a README file"**
5. Click **Create repository**

### 3. Upload the files
1. Inside your new repo, click **Add file → Upload files**
2. Drag and drop ALL these files at once:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Scroll down → Click **Commit changes**

### 4. Enable GitHub Pages
1. Go to your repo's **Settings** tab
2. Scroll to **Pages** (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Branch: **main** / Folder: **/ (root)**
5. Click **Save**
6. Wait ~1 minute, then your URL will appear:
   `https://YOUR-USERNAME.github.io/fuel-tracker/`

### 5. Add to Home Screen

**iPhone (Safari):**
1. Open the URL in **Safari** (not Chrome)
2. Tap the **Share** button (box with arrow)
3. Scroll down → **Add to Home Screen**
4. Tap **Add** — done! 🎉

**Android (Chrome):**
1. Open the URL in **Chrome**
2. Tap the **⋮ menu** (top right)
3. Tap **Add to Home screen**
4. Tap **Add** — done! 🎉

## ✨ Features
- 2,500 kcal daily goal · Protein 190g · Carbs 253g · Fat 81g
- 20 preset foods with full macros
- Add unlimited custom foods (saved locally)
- Adjust serving sizes with stepper / quick presets
- Live macro bars + calorie ring
- **Works offline** after first visit
- Log resets automatically at midnight
- Data saved to your device (localStorage)

---

Just upload new versions of the files to GitHub — the site updates within seconds.
